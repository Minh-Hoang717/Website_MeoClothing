# Meo Clothing Store - Website Bán Quần Áo

## 1. Giới thiệu
Dự án xây dựng hệ thống thương mại điện tử kinh doanh quần áo thời trang.
- **Môn học:** Thực hành Phát triển phần mềm nguồn mở
- **Giảng viên hướng dẫn:** [Tên GV]
- **Nhóm thực hiện:** [Tên Nhóm]

## 2. Công nghệ sử dụng
- **Ngôn ngữ:** PHP (Mô hình MVC tự xây dựng)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, Bootstrap/Tailwind, jQuery

## 3. Cài đặt (Installation)
1. Clone dự án: `git clone https://github.com/[Link-Repo]`
2. Import database từ thư mục `database/meo_clothingstore.sql`
3. Cấu hình file `config/config.php`
4. Chạy server ảo (XAMPP/Laragon) trỏ vào thư mục `public/`

## 4. Quy trình làm việc
Xem chi tiết trong file báo cáo hoặc Wiki của dự án.
